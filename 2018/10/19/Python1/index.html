Python之初识python及一些特殊的地方

发表于 2018-10-19 | 分类于 Python |

HelloWorld!

print('hello world!')

python文件执行

python通过.py执行的时候,需要在文件头部制定解释器 如#!/usr/bin/env python

linux系统中,./xxx.py 代表加权限执行py

中文在各版本中的使用

python3

无需加utf-8,可以识别中文

python2

默认是ASCII编码，不能识别中文，需要在头部加: # -*- coding:utf8 -*-

编码区别

ASCII

用8个二进制数来表示常用字符和英文字母及数字等，总共有256种可能,不支持中文

unicode

用至少16位的二进制数来表示各国语言,缺点是至少16位,有浪费内存的现象

utf8

万国码,对unicode的情况作了优化,使得二进制数的长度可以变化,不会出现浪费内存情况

输入输出

输入

inp = input('输出的语句')

eg:inp = input('请输入用户名:')

终端显示: 请输入用户名:

在终端显示的内容后面需要手动输入内容,所有输入的内容(字符串)将会在Enter之后赋值给inp

输出

print('输出的语句')

条件语句

python if else 必须严格遵守缩进规则（一般缩进一个tab）

if 条件:

    执行语句1

else:

    执行语句2

如果符合条件后不想执行操作,需要pass关键字,否则报错

字符串

引号

name1 = 'zhangsan'

name2 = "lisi"

name3 = '''wangwu'''

name4 = """zhaoliu"""

乘法

n1 = "alex"

n2 = n1 * 10

结果:"alexalexalexalexalexalexalexalexalexalex"

数字

次方

a = b ** c

结果: a等于b的c次方

商的整数部分

a = b // c

结果：a等于b除以c商的整数部分

余数

a = b % c

结果：a等于b除以c的余数
